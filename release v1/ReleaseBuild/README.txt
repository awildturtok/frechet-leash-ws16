Start des Programms:

Zum Starten von frechet.exe muss Qt in der Version 5.8.0 auf dem System installiert sein. 
Dies ist die Vollversion der entwickelten Software.

Des Weiteren liegt ein static build bereit. Die Datei frechet_static.exe kann ohne vorherige 
Installation von QT ausgeführt werden. Diese Version unterstützt jedoch nur die Koordinateneingabe 
über die Texteingabe.
Die Datei muss vor dem Start zu frechet.exe umbenannt werden.
